## 安装项目
npm install -g yarn

设置镜像源：
yarn config set registry https://registry.npm.taobao.org

node-sass镜像源:
yarn config set sass-binary-site http://npm.taobao.org/mirrors/node-sass

安装：
yarn


### 启动项目
yarn start

原生html项目在 src/html 目录下面，此目录下面文件是独立的。
